# MeshPay Relay Network

Offline-first digital payment simulation using Spring Boot and mesh-network transaction relay.

## Features

- Offline payment relay
- Transaction propagation
- Settlement engine
- RSA cryptographic signing
- H2 database
- Interactive dashboard

- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Run Locally

```bash
git clone <repo-url>
cd meshpay-relay-network
./mvnw spring-boot:run
```
